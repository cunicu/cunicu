## wice sync

Synchronize interfaces

### Synopsis

Synchronizes the internal daemon state with the state of the WireGuard interfaces

```
wice sync [flags]
```

### Options

```

  -s, --rpc-socket string   Unix control and monitoring socket (default "/var/run/wice.sock")
```
